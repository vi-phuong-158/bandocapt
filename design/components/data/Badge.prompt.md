Compact category/status label. Solid tones sit over photos (the detail header "Loại đơn vị"); soft tones sit on light cards.

```jsx
<Badge tone="primary">Công an</Badge>
<Badge tone="amber" soft>Điểm CCCD</Badge>
<Badge tone="nearby" soft icon="near_me">1.2 km</Badge>
```

Tones: `primary`, `amber` (CCCD), `nearby` (emerald), `neutral`. For the rounded distance pill specifically, use `Chip`.
