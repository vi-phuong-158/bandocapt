A labelled detail line with a round icon medallion — the address / phone / hours rows in the unit detail panel.

```jsx
<InfoRow icon="location_on" label="Địa chỉ" value="Số 12 Đại lộ Hùng Vương, Việt Trì" />
<InfoRow icon="phone" label="Số điện thoại" value="0210 3845 678" href="tel:02103845678" />
<InfoRow icon="schedule" label="Giờ làm việc" value="T2–T6: 7:30–17:00" />
```

Pass `href` for tappable rows (phone). Stack with ~24px gap.
