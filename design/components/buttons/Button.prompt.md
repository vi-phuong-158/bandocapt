Pill action button in public-security blue — use for primary and secondary calls to action like "Chỉ đường" (directions) and "Gọi điện" (call).

```jsx
<Button variant="primary" icon="directions">Chỉ đường</Button>
<Button variant="soft" icon="call">Gọi điện</Button>
<Button variant="secondary">Đóng</Button>
```

Variants: `primary` (solid blue, glow shadow), `soft` (blue-50 tint — the app's "Chỉ đường" action), `secondary` (slate), `ghost`, `amber` (CCCD). Sizes: `sm` / `md` / `lg`. Pass Material Symbols names to `icon` / `iconRight`. Always fully rounded (pill).
