Design System

Bản đồ Công an số — UI Kit

Hướng dẫn tokens, components, và quy tắc sửa giao diện · Dành cho Claude Code / Codex

Đọc file này trước khi sửa bất kỳ giao diện nào trong index.html, app.js, js/chatbot.js, hoặc file HTML/CSS bất kỳ. Khi sửa UI, không tự chọn màu, font, kích thước, shadow — luôn dùng CSS variables từ design system.

1. Components có sẵn

Button
primary · secondary · ghost · amber · soft. Icon trái/phải.
IconButton
Nút tròn chỉ có icon. Dùng cho FAB, action phụ.
SearchBar
Input pill frosted với leading icon. Focus ring xanh.
FilterTabs
Multi-select filter pills. Công an / CCCD / Gần tôi.
ResultCard
Kết quả tìm kiếm: icon box + tiêu đề + địa chỉ + chip.
Badge
Nhãn loại đơn vị. tone: police · cccd · nearby · warn.
Chip
Filter pill, distance chip. Có trạng thái active.
InfoRow
Hàng thông tin trong detail sheet: icon + label + value.
ChatBubble
Bong bóng chat. role: user · assistant (có avatar).
ChatLauncher
Nút AI nổi với pulsing ring. Avatar mascot icon.png.

8. Quy tắc sửa UI

  • 1
    Màu: chỉ dùng CSS variables. Không dùng #hex trần trong code.
  • 2
    Font: Be Vietnam Pro cho text · Material Symbols Outlined cho icon.
  • 3
    Border-radius: dùng --radius-*. Không hardcode px.
  • 4
    Shadow: dùng --shadow-*. Không tự viết box-shadow.
  • 5
    Touch target: mọi element bấm được phải ≥ var(--hit-min) = 44px.
  • 6
    Glass panels (overlay trên bản đồ): background: var(--glass-fill); backdrop-filter: var(--blur-md).
  • 7
    Animation: dùng var(--dur-fast) / var(--dur-base) + var(--ease-smooth).
  • 8
    Không thêm Tailwind class mới cho UI đã có. Nếu cần thêm style, dùng CSS inline hoặc custom property.
